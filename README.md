# 🚀 HANGIEASY - Viral Quiz & Tournament Platform

**Hangieasy**, kullanıcıların kendi testlerini oluşturabildiği, turnuvalar düzenleyebildiği ve "Kör Sıralama" gibi viral oyun modlarıyla etkileşime girdiği yeni nesil bir eğlence platformudur. 

## 🎯 Özellikler
- **O mu Bu mu? (Versus):** İki seçenek arasından nihai favorini seç.
- **Kör Sıralama (Blind Ranking):** Gelecek öğeyi bilmeden listeni yap, risk al!
- **Turnuva Ağacı (Bracket):** Şampiyonlar Ligi mantığıyla favorini belirle.
- **Viral Paylaşım:** Sonuçlarını tek tıkla WhatsApp ve X'te paylaş.

## 🛠️ Teknoloji Yığını
- **Backend:** Python / Flask
- **Database:** SQLite (SQLAlchemy)
- **Frontend:** HTML5, CSS3, JavaScript
- **Deployment:** Render / GitHub
